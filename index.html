Features

Background & Dashcam Recording

  • Record videos discreetly with screen off for maximum privacy
  • Fragmented MP4 format ensures zero risk of file corruption
  • Auto-splitting for large recordings based on customizable size limits
  • Wide-angle camera detection and support
  • Geotagging with location data embedding

Screen Recording & Annotations

  • FadRec tab for professional screen recording with annotation tools
  • Pen, eraser, text, and shapes for detailed annotations
  • Multi-layer editing with version control and unlimited undo/redo
  • Automatic project saving with full recovery options
  • Sidebar overlay menu for quick-access recording controls

Live Streaming & Remote Control

  • Stream camera feed live over local network with beautiful web interface
  • Remote control: start/stop recording, adjust settings, toggle torch
  • Real-time monitoring: battery level, storage status, recording status
  • Perfect for security monitoring and remote access

Advanced Features

  • Custom watermark text overlays (e.g., license plate, location)
  • Multiple resolutions, 60/90fps support, orientation control
  • 7+ app themes including AMOLED, Light, and System themes
  • 15+ customizable app icons
  • 100% ad-free, open-source, and privacy-focused

Privacy Info

How does FadCam ensure user privacy?
FadCam is designed with privacy in mind. All recordings are stored locally on your device, and no data is sent anywhere.
Is FadCam open-source?
Yes, FadCam is completely open-source. You can view and contribute to the code on our GitHub repository.
Does FadCam have any usage restrictions?
FadCam is intended for ethical use only, such as personal security, research, or documentation. We do not condone any illegal or unethical use of the app.
